```

```

```

```

```

```

## 🎮 How to play<a name = "play"></a>


* Press &#8593; to to move forward 
* Press &#8592; and right.
* Press &#8594; to move left.
* Press &#8595; to move Backward.
* Press `Shift` to increase speed.
